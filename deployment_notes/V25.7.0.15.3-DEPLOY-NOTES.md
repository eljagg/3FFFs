## ⚠️ READ FIRST — about why this release exists

After v25.7.0.15.2 was reportedly deployed, Omar's testing showed
the same audio bugs from v25.7.0.15.1 (IVR unsync, voice cutoffs).
On inspecting the project files Omar uploaded, I confirmed:

**v25.7.0.15.2 was NOT actually live.** The deployed code was still
v25.7.0.15.1 with all the audio bugs intact.

How I verified:
- `audioNarration.js` still had 3 occurrences of `speechSynthesis.cancel()` (v25.7.0.15.2 reduced this to 2)
- `MultiActorSequenceAnimation.jsx` still had `slotDur` slot-distribution code (v25.7.0.15.2 removed this entirely)
- `vishingScenes.jsx` `totalDurationMs` was still `62500` (v25.7.0.15.2 changed it to `81000`)

This means one of these happened with v25.7.0.15.2:
- Zip wasn't unzipped, OR
- Files weren't pushed to GitHub, OR
- Railway didn't pick up the new commit, OR
- Railway built it but cached the old assets, OR
- Browser cached the old JS bundle

**This release adds a runtime version log so post-deploy verification
takes 5 seconds.** Open browser console after deploy. If the new code
is live, you'll see:

```
[3FFFs audio v25.7.0.15.3 — sequential queue]
```

If you don't see this log, or if you see `v25.7.0.15.1`, the new code
did not deploy and the bugs will persist regardless of how many
hotfixes I ship.

## What's in this release

This is v25.7.0.15.2 + OSINT audio + version log. Same audio sync fix:

1. **Architectural fix** in `audioNarration.js`: `speakMessage` no
   longer calls `speechSynthesis.cancel()` per call. Browser's queue
   handles sequencing. Prior bug killed each utterance after 2-3
   words.
2. **Engine fix** in `MultiActorSequenceAnimation.jsx`: replaced
   slot-distribution scheduling with sequential queue at stage entry.
   Browser plays utterances back-to-back.
3. **Visual cue fix**: speaker icon driven by utterance `onstart`/
   `onend` callbacks, not timer offsets.
4. **Script calibration**: every audio script trimmed to fit ≤2.3
   words/second budget within its stage duration.
5. **Stage duration adjustments**: Vishing total 62.5s → 81s; SIM
   Swap 60s → 75s; Phishing 41s → 47s.
6. **OSINT audio added** (NEW in v25.7.0.15.3): all 7 stages of
   Gather Victim Information now have narrator-voice audio.
7. **Runtime version log** (NEW): console outputs version on first
   animation play, eliminating future deploy-verification ambiguity.
