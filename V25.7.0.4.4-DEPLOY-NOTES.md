# v25.7.0.4.4 — Bleed via transform + collapsible techniques

Two fixes:

## 1. Full-bleed via position+transform

`calc(50% - 50vw)` wasn't working — cream dashboard rendered offset right with content clipped left.

Replaced with the more reliable position-based bleed: `position:relative; left:50%; transform:translateX(-50%); width:100vw`. Works regardless of nested padding.

## 2. Collapsible techniques

Reverted the compact-card change from .3. Now the techniques section header is a clickable button — defaults to collapsed, click to expand the cards. Per-tactic state, so opening Positioning techniques doesn't open them for other tactics.
