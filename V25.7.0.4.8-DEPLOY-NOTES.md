# v25.7.0.4.8 — Larger fonts + per-viz collapse toggles

Two changes:

## 1. Larger fonts on the Framework page

Bumped to improve readability of the expanded-tactic body:
- Description: 15 → 17px
- Executive takeaway label: 10 → 11px
- Executive takeaway body: 14 → 16px
- Visualization title: 16 → 18px
- Visualization subtitle: 12 → 13px

NOT changed: the cream Two Views internals (Allison's name, data tables, etc.) and the kill-chain grid internals. Those are self-contained visualization designs with their own carefully-tuned typography. If you want those bumped too, ask separately.

## 2. Per-visualization collapse toggles

Each visualization in the expanded tactic body now has a clickable header that toggles expanded/collapsed.

**Defaults per visualization kind:**
- `positioning_timeline` → expanded (primary teaching visualization)
- `kill_chain_grid` → expanded (Reconnaissance primary)
- `two_views` → collapsed (deep-dive companion; user expands after timeline)

**Click anywhere on the viz title/subtitle to toggle.** Chevron rotates to indicate state.

State is per-visualization. Toggling Positioning's timeline doesn't affect Two Views. Both are independent.

State pattern: storing "deviations from default" rather than absolute open/closed. New visualizations get sensible defaults without backfill — just add an entry to the `VIZ_DEFAULTS` map for any kind that should default-collapse.

## Files changed

- `client/src/pages/Framework.jsx` — font bumps + viz toggle state + new toggle button styles

`Page.jsx` and `PositioningTwoViews.jsx` carry forward unchanged.
