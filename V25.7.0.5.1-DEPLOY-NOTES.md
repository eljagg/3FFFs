# v25.7.0.5.1 — Storyboard fonts hotfix

Single-file font-only patch on top of v25.7.0.5. Bumps every text element in
the scenario storyboard one step larger for readability — the original sizes
were tuned for prototype viewport but undersized in production.

## What changed

`client/src/components/scenarios/ScenarioStoryboard.jsx` — font sizes only,
no logic changes, no API changes, no migration needed.

| Element | Before | After |
|---|---|---|
| Summary card headline | 22 | 24 |
| Summary card meta row | 10 | 11.5 |
| Summary card subtitle | 14 | 16 |
| Summary card stat label | 9.5 | 11 |
| Summary card stat value | 14 | 16 |
| Phase marker | 10 | 12 |
| Day number (D0, D1) | 20 | 22 |
| Day label ("Day 0") | 9.5 | 11 |
| Actor pill (CUSTOMER / CREW / DETECTION) | 9.5 | 11 |
| Beat tech ID | 11 | 13 |
| Beat headline | 16 | 18 |
| Beat narrative | 13 | 15 |
| Detail strip ("BEAT 3" / "DAY 0") | 10 | 11.5 |
| Detail empty state | 13.5 | 15 |
| Detail eyebrow ("CONTEXT BEAT") | 10 | 11.5 |
| Detail title | 22 | 24 |
| Detail ID | 11 | 13 |
| Detail pills (DAY 0 / VICTIM / severity) | 9.5 | 11 |
| Detail section title ("WHAT HAPPENED") | 10 | 11.5 |
| Detail body prose | 14 | 16 |
| Context box ("Why this beat matters") | 13 | 15 |
| Context box label ("In this scenario") | 9.5 | 11 |
| Per-role placeholder | 12 | 14 |

Also nudged line-height and padding on a few elements where the larger fonts
needed slightly more breathing room (actor pill padding, placeholder
line-height, prose line-height).

## Files in this delivery
